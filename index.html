发表于 2020-03-04

hexo博客搭建

1.安装 Node.js + hexo + git

1.1 安装nodejs

nodejs官网 下载LTS版本

node -v

npm -v查看版本

1.2 安装cnpm

npm install -g cnpm --registry=https://registry.npm.taobao.org

cnpm -v 查看版本

1.3 安装hexo：

cnpm install -g hexo-cli

hexo -v

1.4 安装git

git官网 可能需要翻墙 ，git for windows 淘宝镜像 不行就用镜像。

在git bash中的基本设置：

设置用户名: git config --global user.name '你在github上注册的用户名';

设置用户邮箱:git config --global user.email '注册时候的邮箱';

查看是否成功:git config --list

2.创建博客

2.1 创建空文件夹

方法一：直接鼠标右键新建文件夹

方法二：利用git或者命令行mkdir blog

2.2 初始化

然后输入 hexo init 。一个初始的博客就生成啦！

2.3 开启

输入 hexo s开启博客，访问下面的网址，就是初始的博客啦。

可以用 hexo s -p4000 指定端口。

3.部署到Github

3.1 建立仓库

建立与你用户名对应的仓库，仓库名必须为your_user_name.github.io

3.2 修改配置文件

在配置文件的最后改为（把your_user_name改为你的用户名）

type: git
repo: https://github.com/your_user_name/your_user_name.github.io.git
branch: master

3.3 安装插件

cnpm install --save hexo-deployer-git

3.4 部署

输入用户名和密码

4.更换主题

去官网找主题 hexo主题。

可以参考NexT使用。

总结就是在themes文件夹下添加主题文件夹，然后在总配置文件_config.yml配置中更改为theme: next。

关于主题的更改参见NexT使用。

5.后续写文章

超简单~就三步！

然后在source文件夹中更改markdown文章就可以啦。hexo博客搭建
